# Shahriar Imran - Portfolio Website

A modern, responsive portfolio website showcasing competitive programming achievements, projects, and skills.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Single Page Application**: All content on one page with smooth scrolling navigation
- **Interactive Elements**: Hover effects, animations, and mobile-friendly navigation
- **Performance Optimized**: Fast loading with optimized code

## Sections

1. **Home**: Personal introduction with profile picture
2. **Online Contest**: Codeforces profile and achievements
3. **Onsite Contest**: Details of all competitive programming contests
4. **Extra Curricular**: Problem setting, training, and mentoring activities
5. **Skills**: Technical skills and frameworks
6. **Awards**: Academic achievements and certifications
7. **Projects**: Software development projects with GitHub links

## Customization Guide

### 1. Personal Information
Edit the `index.html` file to update:
- Your name in the title and hero section
- Profile picture (replace placeholder image URL)
- Personal description in the hero section

### 2. Profile Picture
Replace the placeholder image URL in the hero section:
```html
<img src="your-actual-image-url.jpg" alt="Shahriar Imran">
```

### 3. Codeforces Profile
Update the Codeforces link in the Online Contest section:
```html
<a href="https://codeforces.com/profile/your-actual-handle" class="btn btn-primary" target="_blank">
```

### 4. Contest Details
All contest information is already included based on your provided data. You can:
- Update contest rankings if they change
- Add new contests by copying existing contest-item structure
- Replace placeholder images with actual contest photos

### 5. Project Links
All GitHub project links are already included. Update if needed:
- Blood Bank Management System
- Online Shop Management System  
- ATM Machine Application

### 6. Skills
Update the skills section if you learn new technologies:
```html
<span class="skill-tag">New Technology</span>
```

### 7. Colors and Styling
The website uses a blue-purple color scheme. To change colors, edit `styles.css`:
- Primary color: `#4a90e2`
- Secondary color: `#764ba2`
- Background gradient in hero section
